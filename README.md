
Для установки и продолжения разработки внутри контейнера, следует выполнить следующие шаги:
1. Установить Docker
2. склонировать репозиторий себе на компьютер
3. открыть терминал в папке с клонированным репозиторием. (можно просто в VS code)
4. ввести команду "docker-compose up -d" после чего в приложении Docker создастся и запустится новый контейнер со всем необходимым для работы. 
5. Чтобы подключиться к окружению python в контейнере необходимо установить расширение в VS Code под названием "Dev Containers"
6. После установки расширения, при запущенном контейнере в VS Code необходимо нажать F1 на клавиатуре и выбрать "Dev Containers:Attach to Running Container..." и подключиться к запущенному контейнеру. При этом появится новое окно VS Code подключенное к контейнеру.
7.Установить в новом окне VS Code расширение Python

Библиотеки используемые в нашем окружении python прописываются в файле requirements.txt с указанием версии.
